Apollo Project

Apollo is a non-profit project with the objective of centralizing data on missing persons in Brazil, using various sources including social networks. Additionally, people registered in the system will be replicated in other regional systems. This system will have notifications and other alerts if someone disappears in the registered user's area.

Apollo uses Microservices in Quarkus, AWS Lambda, Amazon DynamoDB as a Key/Value database, Elasticsearch, RabbitMQ as a messaging service. In the frontend, we use Typescript, Angular, SSR. In both modules we use Docker and Github Actions.
